Barcodes Support
================
Barcode support relies on Barcode4j (http://barcode4j.sourceforge.net/).
If you add the Barocde4j jar to your classpath, Docmosis will pick it up and use it.
You only require the barcode4j.jar file (so other jars in the distro can be ignored).
